# Discord Self-Bot with Key System

## ⚠️ WARNING
This is a **self-bot** which violates Discord's Terms of Service. Use at your own risk. Your account could be permanently banned.

## Features
- 🔑 Key generation and management system
- 📝 Panel setup for each user
- 📨 Scheduled message sending to multiple channels
- 📊 Logging system
- 👥 Multi-user support

## Installation

### 1. Clone or download the files
Download all files to a folder on your computer.

### 2. Install Python 3.8+
Make sure you have Python 3.8 or higher installed.
